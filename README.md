# Face Detection Attendance System

An AI-based attendance system that uses real-time face detection to automate and streamline attendance marking. Built using Python, OpenCV, and face recognition technologies.

## 🚀 Features

- 🎯 Accurate face detection using OpenCV
- 🧠 Face recognition for known individuals
- 🕒 Automatic attendance marking with timestamps
- 📝 Attendance logs saved in CSV format
- 📷 Capture new face data for registration
- 📊 Easy-to-read GUI (optional)
- 💾 Lightweight and offline capable

## 🛠️ Technologies Used

- Python
- OpenCV
- face_recognition (Dlib)
- CSV for attendance records
